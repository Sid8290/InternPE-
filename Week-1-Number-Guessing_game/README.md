# Week 1 - Number Guessing Game

## Objective

Build a Number Guessing Game using Java.

## Project Description

This project is a simple Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the user has to guess the correct number. The program provides hints such as "Too High" or "Too Low" until the correct number is guessed.

## Features

- Random number generation
- User input using Scanner
- Hint system (Too High / Too Low)
- Attempt counter
- User-friendly console output

## Technologies Used

- Java
- Scanner Class
- Random Class
- Loops
- Conditional Statements
