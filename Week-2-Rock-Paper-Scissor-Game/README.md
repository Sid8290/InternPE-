# Week 2 - Rock Paper Scissors Game

## Objective

Build a Rock Paper Scissors Game using Java.

## Project Description

This project is a console-based Rock Paper Scissors game developed using Java. The user competes against the computer by choosing Rock, Paper, or Scissors. The computer generates a random choice, and the winner is determined according to the standard game rules.

## Features

- User vs Computer gameplay
- Random computer move generation
- Winner determination logic
- Input validation
- Interactive console interface
- Multiple rounds support

## Technologies Used

- Java
- Scanner Class
- Random Class
- Conditional Statements
- Loops
