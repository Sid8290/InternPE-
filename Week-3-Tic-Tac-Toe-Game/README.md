# Week 3 - Tic Tac Toe Game

## Objective

Build a Tic Tac Toe Game using Java.

## Project Description

This project is a console-based Tic Tac Toe game developed using Java. Two players take turns placing their symbols (X and O) on a 3x3 board. The game checks for winning combinations and declares the winner or a draw based on the game outcome.

## Features

- Two-player gameplay
- 3x3 game board
- Winner detection
- Draw detection
- Input validation
- Interactive console interface

## Technologies Used

- Java
- Scanner Class
- Arrays
- Loops
- Conditional Statements
